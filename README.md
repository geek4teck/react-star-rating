# Star Rating with Color and number of stars

# Usage

```HTML
<StarRating 
    size= "small | medium | large | 2xlarge | 3xlarge | 4xlarge | 5xlarge"   //Default is small
    n={7} //Number of stars in a row
    color="#FF0000" //Default is #FFCD3C
/>
```
